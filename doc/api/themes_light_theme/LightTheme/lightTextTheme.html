TextTheme? lightTextTheme()

Returns the customized TextTheme for the light theme.

The text theme defines the styling for various types of text in the application, such as headlines, body text, captions, etc. This method provides the customized text theme specifically designed for the light theme.

If the light text theme is not available or not specified, this method returns null.
