Returns true if the given dateTime is yesterday's date.

The dateTime parameter is compared with the current date and time using DateTime.now(). If the year, month, and day of the given dateTime object match the year, month, and day of the current date minus one day, the method returns true, indicating that the given date is yesterday's date. Otherwise, it returns false.

Implementation

bool isYesterday(DateTime dateTime) {
  DateTime currentDate = DateTime.now();
  return dateTime.year == currentDate.year &&
      dateTime.month == currentDate.month &&
      dateTime.day == currentDate.day - 1;
}
