String formatDate(

Returns a formatted string of the date of the given dateTime.

The dateTime parameter is formatted as a string in the format dd/mm/yy. The day and month are padded with a leading zero if necessary using padLeft(2, '0'). The year is formatted as a string and takes the last two digits using % 100. The formatted string is returned.

Implementation

String formatDate(DateTime dateTime) {
  String formattedDay = dateTime.day.toString().padLeft(2, '0');
  String formattedMonth = dateTime.month.toString().padLeft(2, '0');
  String formattedYear = (dateTime.year % 100).toString().padLeft(2, '0');
  return '$formattedDay/$formattedMonth/$formattedYear';
}
