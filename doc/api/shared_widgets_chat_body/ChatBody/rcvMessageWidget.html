Builds the received message widget.

This method returns a Padding widget that contains an Align widget. The Align widget contains a Column widget that displays the message text and a Text widget that displays the message hour. The Container widget uses the isOwnMessage parameter to determine the background color of the message.

Implementation

Widget rcvMessageWidget(double width) {
  return Padding(
    padding: const EdgeInsets.all(10),
    child: Align(
      alignment: Alignment.centerLeft,
      child: Column(
        crossAxisAlignment: CrossAxisAlignment.start,
        children: [
          Container(
            width: width * 0.8,
            padding: EdgeInsets.all(16),
            decoration: BoxDecoration(
              color: Colors.white,
              borderRadius: BorderRadius.all(
                Radius.circular(16),
              ),
            ),
            child: Text(
              text,
              style: Styles.bodySmall,
            ),
          ),
          SizedBox(height: 8),
          Text(
            hour,
            style: Styles.dateTimestamp,
          ),
        ],
      ),
    ),
  );
}
