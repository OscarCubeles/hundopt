Builds the ClickableText widget.

This method returns a GestureDetector widget that contains a RichText widget. The RichText widget displays the text parameter and the uText parameter with different styles. The GestureDetector widget uses the onPressed parameter to determine the callback function when the text is pressed.

Implementation

@override
Widget build(BuildContext context) {
  return GestureDetector(
    onTap: onPressed,
    child: RichText(
      textAlign: TextAlign.center,
      text: TextSpan(
        text: "$text ",
        style: Theme.of(context).textTheme.bodyMedium,
        children: <TextSpan>[
          TextSpan(
            text: uText,
            style: Theme.of(context).textTheme.titleSmall,
          ),
        ],
      ),
    ),
  );
}
